# RE:COIL

A fast-paced action game where recoil is the core movement mechanic.

---

## 🎮 Features

- Recoil-based movement  
- Fast arcade-style gameplay  
- Minimalist visuals  
- Short intense sessions  

---

## 🕹 Controls

- Mouse — Aim  
- Left Mouse Button — Shoot  

---

## 🛠 Built With

- Unity  

---

## 🌍 Platforms

- Windows  
- Linux  
- macOS  
- Web  

---

## 📌 About

This is one of my early released portfolio projects.  
The main focus was gameplay feel, mechanic clarity, and release experience.
